# CSS Guidelines  

---

## 1. Reset Rules  
Always reset margin & padding:  

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box; /* Better control of width & height */
}

## 2. Text Rules 
h1-6 {
    line-height: 1.2;
    letter-spacing: -2px;
}

p {
    line-height: 1.5;
}

a{
    text-decoration: none;
}
